How to Organize and Analyze Research Data

This short lesson is intended for graduate and advanced undergraduate students undertaking original research projects. After completing the 30-minute lesson, you will be able to:

  • Format raw research data in an Excel spreadsheet for efficient analysis
  • Run basic descriptive statistics in Excel
  • Prepare the document for statistical analysis in the software package PASW

Lesson Requirements

For this lesson, you'll need the following:

  • Microsoft Excel
  • PASW (formerly SPSS) statistics software, or free trial download

You should already be familiar with:

  • The basic functions and tools of Excel
  • Descriptive statistics and inferential statistics
  • Research design and how to identify independent and dependent variables

Format Data for Analysis

Organize and Format Categorical Data

Ensure consistency by standardizing data values. For example, change "female" to "F" and reformat multi-component data into individual columns.

Replace categorical data values with "0" or "1" to create dichotomous variables.

Dichotomous Variables

Dichotomous variables allow for the easy isolation of characteristics for analysis, enabling comparisons across experience levels and other categories in PASW.

Running Descriptive Statistics

Copy the Raw_Dataset values into Sheet2 and rename it Formatted_Data. Add columns for categorical values and replace data values with "0" or "1" where needed.

To run descriptive statistics:

  1. Go to Tools > Data Analysis and select Descriptive Statistics.
  2. Enter the input range and check Summary Statistics.

Insight from Descriptive Statistics

Descriptive statistics can reveal interesting patterns, such as variations in perc_enjoy and perc_usefulness. High variance in perc_usefulness might indicate mixed perceptions of the system's utility.

Running Inferential Statistics

Open the dataset in PASW and select Analyze > Nonparametric Tests > K Independent Samples to run the Kruskal-Wallis test for the perc_usefulness and free_play variables.

Once the analysis is complete, save the output as a separate document for easy reference.

Choosing the Correct Statistic

Consult UCLA's Choosing the Correct Statistic guide to ensure that the statistical analysis aligns with your research question and data types.
